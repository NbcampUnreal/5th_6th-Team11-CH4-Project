# Bio-Protocol
연구소에 치명적인 바이러스가 퍼져 모든 직원이 감염되었습니다. 중앙 관리 AI 는 바이러스 유출을 막기 위해 시설 폐쇄(Lockdown)를 결정하고, 인간과 똑같이 생긴 청소부 안드로이드(Cleaner)들을 직원 틈에 잠입시켜 오염원(인간) 제거를 명령합니다.

(사진/영상)

## 개요
- **장르 / 목적** : 1인칭 SF 서바이벌 마피아
- **플랫폼** : PC (Windows)
- **개발 기간** : 25/12/4 ~ 26/01/01
- **엔진/기술 스택** : UE5 5.4 / C++ & Blueprint
- **협업 도구**: GitHub / Slack / Notion 

## 팀원 소개
박태웅:팀장,기획,게임시스템,UI/HUD

김대호:기획,레벨 디자인,상호작용 액터

전민성:멀티플레이 시스템,보이스채팅

강병권:캐릭터, 전투, 상호작용 액터

## 주요 기능
- ✔️ 최대6인 Dedicated 멀티플레이 및 EOS를 이용한 음성 채팅
- ✔️ 캐릭터 스킨 선택
- ✔️ 밤,낮 페이즈 전환
- ✔️ 도구 사용를 사용하여 탈출포드 활성화 및 탈출
- ✔️ 격리실 감금 및 구출
- ✔️ AI 안드로이드 스킬

## 게임 플로우 및 승리 조건

### 게임플로우
#### 낮 - 업무 및 파밍(450초)
직원: 미션 오브젝트를 찾아다니면 알맞는 도구로 미션수행을 하여 탈출포드 활성화<br/>
AI: 직원 인척하며 미션수행을 방해하며 탈출포드 활성화를 방해함<br/>

#### 밤 - 생존 및 전투(45초)
밤이 되면 맵 전체가 소등<br/>
직원:AI를 피해다니며 생존<br/>
AI:전투 모드로 변신 할수있으며 전투모드시 스킬들을 이용해 직원들을 찾아다니며 죽이기<br/>

#### 격리 및 구출
직원,AI 구분않고 체력이 0이 될시 각각 독방 격리실로 이동<br/>
격리 된채로 60초가 지나면 소각->관전자모드로 전환<br/>
격리된지 60초가 되기전에 살아있는 직원이나 AI들이 격리된 유저 구출 가능
